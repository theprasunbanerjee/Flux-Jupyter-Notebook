# 🎨 Flux — Jupyter Notebook Playground

A Jupyter Notebook for experimenting with **Flux**, the state-of-the-art text-to-image generation model.  
Explore image generation, prompt engineering, and model parameters interactively.

---

## 🧠 What It Does

- Loads the Flux model in a Jupyter environment
- Lets you run text-to-image generation experiments
- Provides a scratchpad for tuning prompts, steps, guidance scale, and seeds
- Saves generated images for comparison

---

## 🚀 How to Run

```bash
git clone https://github.com/theprasunbanerjee/Flux-Jupyter-Notebook.git
cd Flux-Jupyter-Notebook

pip install -r requirements.txt   # if present
jupyter notebook
```

> **Note:** A GPU is recommended for fast inference. Works on CPU but is slow.

---

## 🛠️ Tech Stack

- Python 3.x
- Jupyter Notebook
- Diffusers / Transformers (Hugging Face)
- PyTorch
